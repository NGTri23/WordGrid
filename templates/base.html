Guide du jeu

Objectif

Trouvez le mot secret en un nombre limité de tentatives. Après chaque essai, les lettres changent de couleur pour vous indiquer si vous êtes sur la bonne voie.

Comment jouer

  1. Saisissez un mot dans la grille.
  2. Validez avec la touche Entrée.
  3. Observez les couleurs et ajustez votre prochaine tentative.

Signification des couleurs

P
O
I
N
T
A

Bonne lettre, bonne position

La lettre est dans le mot et correctement placée.
A

Bonne lettre, mauvaise position

La lettre est dans le mot, mais à un autre emplacement.
A

Lettre absente

La lettre ne figure pas dans le mot.

Modes de jeu

Normal

Le mode classique. Devinez le mot en un nombre limité de tentatives, sans contrainte de temps.

Chronométré

Même règles que le mode Normal, mais votre temps de résolution est enregistré. Les meilleurs scores apparaissent dans le classement sur la page d'accueil.

Duo (1v1)

Deux joueurs s'affrontent sur le même appareil, chacun devant trouver son propre mot. Le premier à trouver la solution remporte la partie.

Règles

  • Le mot à deviner comporte un nombre fixe de lettres.
  • Vous disposez d'un nombre limité de tentatives.
  • Chaque proposition doit être un mot valide reconnu par le jeu.
  • Le mot respecte la langue et le thème sélectionnés en début de partie.

Bonne chance !
